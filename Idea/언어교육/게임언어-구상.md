> [!quote] 한 문장
> **같이 게임하며 팀을 짜서 외국어로 소통 — 승부욕으로 발화를 끌어올린다.**
> 언어 = "배워야 할 것"이 아니라 **"이기기 위한 도구".** 안 쓰면 진다 → 필사적 사용 = 실전 체화.

> ⚠️ **만드는 게 아니라 호스팅.** 자체 게임 개발 X — 기존 게임 + 언어 룰 + 채연 진행. (IMMERSE·디스코드 이미 검증)

---

## 1. 왜 강력한가

| 일반 학습 | 게임 협동 |
|----------|----------|
| "배워야 해" 의무 | **"이기고 싶어" 욕구** |
| 표현 외우기 | 안 쓰면 짐 (필연) |
| 혼자·평가 불안 | 팀 + 게임이라 실수 OK |

→ 인생도형 **몰입(y) 정점.** 동기·몰입·실전이 한 번에. 채연 場(가상 공간)·환대(팀) 정합.

---

## 2. ★ 3축 게임

| 게임 | 유형 | 언어 강제 | 강점 |
|------|------|:-:|------|
| **마인크래프트** | 협동 빌드·탐험 | 중 | 자유도·재미·場 ★ |
| **마피아 (인랑·마피아42)** | 토론·추리 | **높음** | 설득·변명 회화 강제 ★★★ |
| **어몽어스 / Goose Goose Duck** | 임포스터 찾기 | **높음** | 토론·거짓말·승부욕 ★★★ |

### 축별 성격

```
마크   = 협동·빌드 (함께 만들며 소통) — 부드러운 몰입
마피아 = 토론·설득 (말 안 하면 짐) — 발화 강제
어몽   = 추리·변명 (필사적 변명) — 회화 폭발
```

→ **빌드(마크)는 자유·재미 / 마피아·어몽은 *말하기 강제*.** 회화 학습엔 마피아·어몽 우위, 몰입·관계엔 마크.

---

## 3. ★ 언어를 "이기는 도구"로

```
팀 미션 (빌드 / 마피아 색출 / 임포스터 찾기)
  ↓ 성공 조건
외국어 소통 필수 (룰: "일본어로만")
  ↓
승부욕 → 표현 필사적 사용
  ↓
실전 체화 (의무 X, 욕구)
```

| 게임 | 강제 발화 예 |
|------|------------|
| 마크 | "ここに木を置いて" (여기 나무 놔줘) — 협동 지시 |
| 마피아 | "私は村人です、信じて" (나 마을사람이야, 믿어) — 변명·설득 |
| 어몽 | "あの時、君と一緒にいた" (그때 너랑 같이 있었어) — 알리바이 |

→ 표현이 *시험*이 아니라 *무기*. "변명 못 하면 죽음" = 필연 학습.

---

## 4. 기존 도구 활용 (만들 필요 X)

| 도구 | 용도 | 부담 |
|:-:|------|:-:|
| **디스코드 음성** | 모든 게임 소통 채널 | 최소 ⭐ |
| 마크 서버 | 협동 빌드 | 중 |
| 어몽/Goose Goose Duck | 무료·즉시 | 최소 |
| 게더타운·WorkAdventure | 가상 공간 (건재) | 중 |
| (참고: IMMERSE) | VR 호스트 게임밤 — 검증 모델 | — |

→ 채연 = *개발자* X → **"호스트·기획자".** 기존 게임 위에 *언어 룰·진행·문화*만. 게더타운은 안 죽었고 대안도 풍부.

---

## 5. 현실 진입 (가벼움 순)

| 단계 | 형태 |
|:-:|------|
| **MVP** | 디스코드 음성 + 어몽/Goose Goose Duck + "일본어로만" 룰 |
| 1 | 주 1회 게임 세션 (교환수업 멤버) |
| 2 | 마크 협동 빌드 추가 (場·관계) |
| 3 | 레벨별 룰 (초보=단어 OK, 고급=문장 강제) |

→ **무료·즉시.** 자체 개발 0. [[한국어교환수업-구상]] 멤버 = 첫 플레이어.

---

## 6. ⚠️ 현실 제약 (정직)

| 제약 | 대응 |
|------|------|
| **실시간 = 시간 잠식** | 주 1회 이벤트만 (콘텐츠는 비휘발·자동, 게임은 노동) |
| 레벨 매칭 | 초보=단어 허용, 팀에 중급 섞기 |
| 채연 게임 친숙도 | 진행자 역할 — 직접 고수일 필요 X |
| 밀도 ↓ (실시간) | **주력 X, 커뮤니티 활성 이벤트** |

→ 인생도형: 차별·몰입 ★★★★★ / 밀도(비휘발) ★★ (실시간이라). **콘텐츠(밀도)와 반대 성격 — 이벤트로 보완.**

---

## 7. 위치 — 주력 아닌 "몰입 이벤트"

| | 주력 (밀도) | 이벤트 (몰입) |
|---|------------|--------------|
| 콘텐츠·덱 | ★★★★★ | — |
| 교환수업 | ★★★★ | — |
| **게임 세션** | — | ★★★★★ |

→ **게임 = 가끔 하는 *몰입·재미·관계* 이벤트.** 콘텐츠가 *학습 자산*이면, 게임은 *커뮤니티 활성·동기 점화*. 둘이 보완.

---

## 8. 수익·정합

| 모델 | |
|------|---|
| 무료 | 게임 세션 참여 (커뮤니티 활성·유입) |
| 후원·멤버십 | 정기 게임밤 우선 참여 |
| 교환수업 연계 | 팀 = 다국어 학습자 → 상호 교수 게임화 |

→ 비상업 정신 (관계 무료·운영 후원). 게임 = *유입·활성 미끼* → 콘텐츠·교환수업 수익으로.
