> 🦴 **뼈(레벨별 표현)가 척추, 살(감각·문화)이 그 위.** 표현 먼저 정의 → 맥락(장소·오감)을 입힌다.
> 일본어 기준 (영·중·한 복제). **뼈 = 흔들림 0, 살 = 채워넣기.**

---

## 1. ★ 뼈·살 원칙

```
뼈 (레벨별 표현)  → 학습의 척추 (필수·체계·"배운다")
   + 살 (장소·문화·오감·작품)  → 몰입 (재미·기억·"빠진다")
   = 콘텐츠 1편
```

| 순서 | 뼈 먼저 → 살 나중 |
|:-:|------|
| ❌ 옛 방식 | 장소 정함 → 표현 끼움 (살이 뼈 덮음 = 야네센 문제) |
| ✅ 새 방식 | **표현 정함 → 그 표현 쓰는 장소·살 입힘** |

> 뼈 또렷(1~2개) + 살은 *선택*. 어려운 단어(揚げたて 등)는 살이라 *빼도 됨* — 뼈(ください)만 필수.

---

## 2. ★ 레벨 (L0~L3, Can-do 기준)

> JLPT 급수 X → **"이걸 할 수 있다"** (행동 가능). 학습자가 자기 위치 앎.

| 레벨 | Can-do | 글자 | JLPT |
|:-:|------|:-:|:-:|
| **L0 입문** | 히라가나·가타카나 읽기 + 인사 | 배우는 중 | — |
| **L1 생존** | 자기소개·주문·질문·숫자 | 읽기 가능 | ≈N5 |
| **L2 일상** | 길·취미·감정·과거 이야기 | +한자 초급 | ≈N4 |
| **L3 자연** | 의견·뉘앙스·존댓말·문화 대화 | +한자 중급 | ≈N3 |

→ **L0 (글자)가 척추의 시작.** 야네센이 "맨 처음" 막힌 건 L0가 없어서.

---

## 3. ★ 표현 뼈대 맵 (척추)

> 각 레벨의 *필수 표현*. 콘텐츠는 이 뼈를 하나씩 살로 입힘.

### L0 — 입문 (글자·인사)

| # | 뼈 | 살 (예) |
|:-:|------|------|
| 0-1 | 히라가나 (あ행~) | 글자 유래·리듬 |
| 0-2 | 가타카나 | 외래어·간판 |
| 0-3 | 인사 (こんにちは·ありがとう·すみません) | 일상 장면 |

### L1 — 생존 (です/ます 체계) ⭐

| # | 뼈 | 살 (예) |
|:-:|------|------|
| 1-1 | 자기소개 (〜です·よろしく) | 첫 만남 |
| **1-2** | **주문·요청 (〜ください)** | **[[야네센]] 멘치카츠** ⭐ |
| 1-3 | 질문 (〜ですか) | 길묻기·다방 |
| 1-4 | 공감 (〜ね·〜よ) | 노을 계단 |
| 1-5 | 숫자·가격 (〜円·いくら) | 상점 |

### L2 — 일상

| # | 뼈 | 살 (예) |
|:-:|------|------|
| 2-1 | 과거 (〜ました·でした) | 여행 회상 |
| 2-2 | て형 (〜てください·〜ている) | 길 안내·동작 |
| 2-3 | 욕구 (〜たい·〜がほしい) | 취미·쇼핑 |
| 2-4 | 이유 (〜から·〜ので) | 의견 |

### L3 — 자연

| # | 뼈 | 살 (예) |
|:-:|------|------|
| 3-1 | 경어 (〜ていただく·尊敬語) | 격식·비즈니스 |
| 3-2 | 뉘앙스 (〜かもしれない·〜はず) | 추측·대화 |
| 3-3 | 접속·복문 | 문학·사회 |
| 3-4 | 관용·문화 표현 | 위인 산책·미의식 |

→ 표현(뼈)이 *레벨을 정의*. 콘텐츠 = 뼈 1~2개 + 살.

---

## 4. 콘텐츠 단위 = 뼈 1 + 살

```
콘텐츠 1편 = 표현 뼈 1~2 (레벨 정의)
           + 살 (그 표현 쓰는 장소·오감·문화)
```

| 야네센 = L1-2 | |
|------|---|
| 뼈 | **ください (주문)** — 또렷이 1개 중심 |
| 살 | 야네센 멘치카츠 가게 (맥락·오감·시타마치) |
| 선택 살 | 〜ね·문화 (덤, 필수 X) |
| 빼기 | 揚げたて 등 어려운 단어 (뼈 덮음 → 선택/제거) |

→ 뼈는 *필수·또렷*, 살은 *몰입·선택*. 학습자는 *최소 뼈*만 가져가도 성공.

---

## 5. 콘텐츠 ↔ 레벨 매핑 (정정)

| 콘텐츠 | 레벨 | 뼈 |
|------|:-:|------|
| (없음 — 추가 필요) | L0 | 글자·인사 |
| 첫 만남편 | L1-1 | 자기소개 |
| **야네센** | **L1-2** | **ください** |
| 길묻기편 | L1-3 | ですか |
| 위인·문화 | L3 | 관용·뉘앙스 |

→ **야네센 = L1-2 (글자 아는 초급).** L0(글자)가 *앞에* 있어야 0→야네센 연결.

---

## 6. 시리즈 축 (살의 종류 — 레벨 무관)

| 살 종류 | 배치 |
|:-:|------|
| 여행 시뮬·장소 컬렉션 | 전 레벨 (살의 기본) |
| 번역 안 되는 말 | L2~ |
| 한 곡으로 | L2~ |
| 계절 | 전 레벨 |
| 위인 산책 | L3 (깊이) |
| 내 동네 번역 | L2~ (산출) |

→ 같은 뼈(표현)에 *다양한 살*을 입혀 시리즈 확장. 레벨 올라갈수록 살(문화·인물) 비중 ↑.

---

→ **뼈(레벨별 표현 맵) = 척추 / 살(장소·문화·오감) = 그 위.** 표현 먼저, 맥락 나중. 야네센 = L1-2(ください). L0(글자)부터 척추 시작. ([[_템플릿]]으로 각 편 = 뼈1+살)
