# RFC — 커리큘럼 선행 관계 v1 (프랑스어 A1 우선)

상태: 채택(2026-07-24). #583·R2의 선행 관계 지적(장면의 미학습 문법 선사용·순서
비자연·연결 데이터 부재) 이행 설계. 원칙: **slug·진도 키 불변, order와 메타만 조정**
(order는 표시 순서일 뿐 저장 키가 아님 — slug-migration RFC에서 확립).

## 1. 메타 필드 (스키마 확장 — R1)

- `prerequisites: [slug, ...]` — 이 챕터가 전제하는 챕터. 코스 지도가 권장 경로
  표시에 사용(강제 잠금 아님 — 성인 자기주도 원칙).
- `formulaic: true` — 장면 고정구 챕터 표지("여기 나온 문형은 통암기 대상,
  분석은 후속 챕터" 안내를 렌더가 자동 표시). 장면 21~28에 부여.
- contentOverrides CHAPTER_KEYS에 두 필드 추가, 계약 테스트에 유효성(존재 slug 참조).

## 2. A1 order 재배열 (프랑스어 — R2)

권장 흐름(#583): 생존 인사 → 소리 최소 규칙 → 주어/être/avoir → 성·관사·복수 →
규칙동사 → 부정·의문 → 형용사·소유 → 이동·부분관사 → 수·시간 → 확장 → 장면.

- 발음 3챕터(a1-29~31)·생존(a1-13)·복수(order 16)를 권장 흐름 위치로 order 재부여.
- slug는 그대로(불일치는 무해 — 확립 원칙), 재배열 후 order 유일성·연번 검사.

## 3. 표시 (R3)

- 코스 지도(/learn/course): prerequisites 미완료 챕터에 '먼저 보면 좋아요' 배지,
  formulaic 챕터에 '장면 고정구' 배지. 잠금은 하지 않는다.

## 4. 순서·분담

R1 스키마(codex 발주 가능) → R2 fr A1 order 재부여+prerequisites 데이터(Claude 정본)
→ R3 코스 지도 배지(codex 발주 가능). ja·en·zh 선행 관계는 fr 정착 후 동일 패턴.
