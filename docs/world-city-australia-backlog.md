# 호주 도시맵 백로그 (오너 지정 2026-07-16 — "브리즈번(선샤인/골드 코스트)·시드니·캔버라·멜버른·애들레이드·퍼스·다윈 추가")

착수 웨이브 미배정 — 저장만(수도권 위성 백로그와 동일 지위). 현 확정 큐(부산·서울 → 유럽
1차 → 런던권 → 유럽 2차) 뒤에 오너 픽으로 편성. APAC 오버월드 bbox[60,-47,180,61]가 호주
전역을 이미 포함하므로 지역 신설 불요 — 도시맵 + 오버월드 노드(공항 게이트)만 얹으면 된다.

## 시스템 접점
- **영어 트랙 기존재** — 런던권과 같은 도항 게이트 축. 영어 하나로 영국+호주 두 대륙 시연
  ("프랑스어 하나로 여러 나라"의 영어판). 콘텐츠 2축: [설정 언어 <en-AU>].
- 남반구 첫 도시들 — 계절 반전(1월=여름) 소재가 시간대/계절 시스템에 걸리면 첫 실전 무대.
- 항공 허브: 시드니(SYD)·멜버른(MEL)·브리즈번(BNE)·퍼스(PER) — 인천/나리타 직항 실존,
  오버월드 항공 게이트 계약 재사용. 퍼스는 런던 직항(QF9)이 있어 EMEA 연결 소재.
- 도시 간: 시드니~멜버른 XPT 야간열차·시드니~캔버라 철도 실존 — "직접 하차" 원칙 유지.
