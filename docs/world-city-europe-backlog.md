# 유럽 도시맵 백로그 (오너 구상 2026-07-16 — 착수 금지, 저장용)

전제: 콘텐츠 수위는 오너 결정 대기 — (a) 지형+교통 / (b) 관광 desc 포함(권장: 2축 구조로 프랑스어
학습이 나중에 gloss 축만 얹으면 됨). 착수 순서는 부산→서울 이후 오너 픽.
프랑스어 트랙 데이터(a0~c1)가 이미 앱에 존재 — 프랑스어권 우선의 근거.
파리는 오버월드에 CDG 항공 게이트·철도 허브 기존재.

## 줌 티어 구조 (오너 확정 방향: "가까이 갈수록 더 가까이 본다")
오버월드 4.5km/타일 → 도시맵 20m/타일 → **마을 정밀맵 4m/타일**(신규 티어).
- 정밀 티어 기준: 본질이 골목·마당 스케일(20m 타일이 집 한 채를 삼키는 곳).
- 진입: 상위 맵 POI에서 게이트(캐널시티 "안쪽 들여다보기"의 일반화). CityScene 엔진 재사용
  (m/타일은 생성 파라미터). 초소형(1.5만~6만 타일)이라 비용 낮음, 단 OSM 건물 폴리곤+수동
  오버라이드(버전 고정 패치) 비중 증가.

## 묶음 원칙
목표 지점들이 10~20km 안이면 1맵(연담 관광권), 멀면 초소형 개별 맵+교통 게이트, 단일 스팟이면
스토리 씬(두바이 라운지 문법). 오버월드가 전역을 담고 있으므로 도시맵 = "확대경 배치" 문제.

## 프랑스 (추천 순서: 파리 → 몽생미셸 → 니스권 → 마르세유)
- **파리**(단독): 에펠탑·루브르·노트르담·개선문·몽마르트르·마레·라탱. 시내+숲 ≈0.3M(베르사유
  포함 시 0.9M — 별도 소형맵 옵션).
- **몽생미셸**(초소형 + 정밀 티어 1호 후보): 섬 전체. 밀물·썰물 갯벌 보행로 시스템 여지.
- **코트다쥐르 묶음**(1맵 ~1.1M): 니스+앙티브(피카소)+생폴드방스(샤갈, 정밀 티어 후보)+에즈(절벽
  마을, 정밀 티어 후보).
- **마르세유**(단독 ~0.4M): 구항·파니에 — 항만·페리 문법 재사용(부산 포지션). 엑상프로방스와
  25km — 묶음 가능하나 성격 상이(옵션).
- **리옹**(보류): 미식 수도 — 경제 RFC 직업(요리) 시스템 열릴 때 최고 무대.
- **루아르 심장부 묶음**(~0.15M): 앙부아즈(클로 뤼세 — 다빈치 최후)+슈농소. 샹보르는 40km 밖(선택).
- **예술가 순례 마이크로**: 지베르니(모네 정원 — 정밀/스토리 씬), 오베르쉬르우아즈(고흐 최후),
  바르비종(밀레). 파리에서 기차 게이트 당일치기 — 고흐 생애 동선(아를→오베르) 스탬프 축.
- **아를**(단독 소형): 고흐의 남프랑스 + 로마 원형경기장(세계유산). **엑상프로방스**(단독 소형):
  세잔·졸라. (아를~엑상 65km — 묶음 비추: 빈 농촌 3M+ 낭비)
- **낭트**(쥘 베른), **루앙**(모네 연작·플로베르·잔 다르크): 단독 소형.

## 프랑스어권 (프랑스 밖)
- **브뤼셀**(~0.3M): 그랑플라스(세계유산)·마그리트·EU 지구 — **다언어 복수 앵커(fr/nl) 첫 실전**
  (헌장 §3-8).
- **제네바**(~0.15M): 레만호 제트분수·UN·루소(출생지)·볼테르(페르네).
- **레만호 연안 묶음**(~0.3M): 로잔~브베~몽트뢰~시옹성 + 라보 포도밭(세계유산) — 호반 선형.
- **모나코**(초소형): 몬테카를로·F1 시가지 — ⚠️카지노는 명칭·외관 지리 참조만(내부·도박 재현 금지).
- **룩셈부르크**(초소형): 구시가 요새(세계유산).
- 장기: 몬트리올·퀘벡시티(아메리카 지역 신설 시).

## 정밀 티어(4m/타일) 후보 종합 — 유럽 밖 포함
몽생미셸 / 생폴드방스·에즈 / 지베르니 / **감천문화마을**(부산 맵 내) / **북촌한옥마을**(서울 맵 내) /
하회마을(안동) / **시라카와고**(일본 신규 — 갓쇼즈쿠리) / 츠마고·마고메(나카센도)·이네 후나야(장기).

## 시스템 접점 메모
- 도항 게이트: 프랑스어 기초 수료 → 프랑스+브뤼셀+로망드+모나코+룩셈부르크 = "프랑스어 하나로
  여러 나라" 첫 시연.
- TGV·탈리스 축(파리–브뤼셀, 파리–제네바, 파리–리옹–마르세유·니스)이 기존 유럽 철도 허브 계약에
  자연 연결. 인물 콘텐츠는 역사적 인물 사실 desc 중심, 미술관·작품은 외관·명칭 참조(내부 재현 금지).
