## 2. 시각 인벤토리 — 텍스처·props·스킨

### 2.1 타일 스킨 현황

| 종류 | 설정값 | 사용 지구 | 의도 |
|------|------|--------|------|
| **building** | `terracotta` | 전체 | R4 구시가 오렌지 지붕 톤 |

**수치가 말하는 격차**:
- 타일 스킨은 1종만 설정: 모든 건물 파사드가 테라코타 색상.
- 시각적 단색화 위험: 지구별 시각 차이를 props·NPC·발견 텍스트로만 구분.

### 2.2 Props 종류 및 배치

| Props | 종류 | 개수 | 배치 위치 | 함수 |
|--------|------|------|---------|------|
| 분수 | fountain | 1 | 테로 광장 [169, 179] | 광장 상징 |
| 좌판 | stall | 1 | 레알 시장 [230, 202] | 시장 활동 |
| 파라솔 | parasol | 1 | 테트도르 호숫가 [242, 121] | 해변 휴식 |
| **합계** | 3종 | 3개 | — | — |

**수치가 말하는 격차**:
- Props 총 3개: 각 지구당 평균 0.19개 — 극도로 희소.
- 종류는 3가지로 다양하나 총량 부족 → 시각적 채움감 낮음.
- 랜드마크 시각화 부족: 푸르비에르 대성당·콘플루앙스 강 합류점 등 상징 props 미배치.

### 2.3 노드 종류 분류

| 카테고리 | 개수 | 종류 |
|---------|------|------|
| **NPC** | 5 | 카페 종업원·구시가 주민·시장 상인 3종 |
| **Spot (발견)** | 11 | 모두 `facade: 'sign'` (단색 팻말 렌더) |
| **문 (도어)** | 2 | `fr-19`(식료품 시장)·`fr-20`(카페 테라스) |
| **합계** | 16 | — |

**수치가 말하는 격차**:
- 모든 발견이 `facade: 'sign'` — 건물 입구 일괄 팻말 스타일.
- 특정 랜드마크(대성당·광장 분수·박물관)의 전용 facade 부재 → 시각 단조.
- NPC와 발견 비율 1:2.2 — NPC 상호작용 충분하지만 정보성 발견은 많음.

## 4. 루프 인벤토리 — 행동·보상·10분 시나리오

### 4.1 가능한 행동 목록

| 행동 | 설정 | 보상 | 현황 |
|-----|------|------|------|
| **이동** | 기본 | 타일 밟기·음향 피드백 | ✓ |
| **NPC 대화** | 5개 노드 | 학습 UI 진입·음성 더빙 | ✓ (3종만) |
| **발견** | 8개 factLine | 텍스트 표시·카운트 증가 | ✓ |
| **도어 학습** | 2개 트랙 | 콘텐츠 재생·스탐프 | ✓ (법정 발주) |
| **역 진입** | 2개 | 다음 구간 진입·명판 | ✓ |
| **전철** | 1개 라인 | 이동 단축 (미구현) | ⚠️ 비활성 |
| **수첩** | UI | 진행 추적 | ✓ (학습 페이지 연동) |
| **스탐프** | 위 모든 행동 | 도장 수집 | ⚠️ `noStamp=true` (비활성) |

**수치가 말하는 격차**:
- 스탐프 비활성 (`noStamp: true` 전체): 게임 루프의 수집 동기 약화.
- 전철 상호작용 미구현: UI 역할만 함.
- NPC는 5개이나 **도어 2개만 학습 진입 가능** — 콘텐츠 소비 경로 좁음.

### 4.2 보상 체감 요소

| 행동 | 토스트 | 카운트 증가 | 칭호/뱃지 | 진행도 |
|-----|--------|----------|---------|--------|
| **발견** (8회) | "발견!" 시각 신호 | +1 발견 카운터 | — | guidebook 진척도 +1 |
| **도어** (2회) | 콘텐츠 UI 진입 | — | 스탐프 (비활성) | 학습 진행도 |
| **NPC** (5회) | 대화 위젯 | — | — | 학습 진행도 |
| **역** (2회) | — | — | — | 경로 진척 |

**수치가 말하는 격차**:
- 즉시 피드백: 발견·도어만 (10개 지점).
- 누적 동기: 스탐프 7개 중 실제 활성은 0개 — **보상 시각화 부재**.
- 10분 플레이 시 예상 이벤트: **4개 발견** (전체 508타일 중 1000타일 추정) = **0.30% per tile**.

### 4.3 10분 플레이 시나리오 추정

```
도시 진입 → 페라슈 역 (시작)
  ↓ 89타일 (약 3분)
  발견-d1 (빵집)
  ↓ 진행 38타일
  발견-d2 (강) → 구시가 진입
  ↓ 진행 32타일
  발견-d3 (트라불) → 푸르비에르 언덕
  ↓ 진행 78타일 (가장 긴 구간)
  발견-d4 (합류점) → 테로 광장
  
예상 10분: ~1000타일 커버 = 처음 4구간 완주 (4개 발견)
실제: 상호작용·카메라 이동·UI로 더 느림 = 2~3개 발견만 경험
```

**수치가 말하는 격차**:
- 콘텐츠 밀도: **0.30% per tile** — 매우 희소 (평균 300타일마다 1개 이벤트).
- 초기 플레이 체감: "넓은 도시 속 조용한 여행" (의도적) vs. "할 게 없다" (위험).
- **메인 루프 의존성**: 발견 8개 + 도어 2개 + NPC 5개 = **총 15개 상호작용점**.

## 5. 종합 격차 분석

### 5.1 핵심 수치 요약

| 항목 | 수치 | 평가 |
|------|------|------|
| **콘텐츠 밀도** (100타일당) | 3.97% | 낮음 — 주 활동 지구만 5~7% |
| **최장 무콘텐츠 구간** | 135타일 (크루아루스→할레스) | 중간 — 플레이타임 2~4분 |
| **텍스처 종류** | 1종 (테라코타) | 매우 낮음 — 지구 시각 분화 부재 |
| **Props 배치** | 3개 (0.19개/지구) | 매우 낮음 — 환경 채움감 부족 |
| **mainRoute 발견** | 8개 (0.5/100타일) | 높음 — 규칙적 몰입감 |
| **도어 (학습 진입)** | 2개 | 낮음 — 콘텐츠 소비 경로 제약 |
| **10분 이벤트** | 2~4개 | 낮음 — 초기 플레이 지루함 위험 |
| **스탐프 활성화** | 0/7 (`noStamp=true`) | 극도로 낮음 — 수집 동기 제거 |

### 5.2 출시 위험 신호

1. **콘텐츠 캐러반 vs. 빈 거리**: 최대 밀도 지구(프레스킬 7.09%) vs. 최소(콘플루앙스 0.88%) = **8배 차이**. 신입 유저는 "넓지만 텅 빈" 느낌 받을 수 있음.

2. **시각적 단조화**: 타일 스킨 1종, props 3개 → 모든 지구가 시각적으로 유사. 지구별 차별화는 발견 텍스트에만 의존.

3. **스탐프 상실**: `noStamp=true` 전체 적용으로 수집 루프 차단. 일본·한국 라운드와의 게임성 낙차 발생.

4. **도어 콘텐츠 병목**: 2개 도어만 학습 콘텐츠 제공. 프랑스 교육 경험이 매우 제한적.
