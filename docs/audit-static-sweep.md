## A. 즉시 결함 (코드로 확정 — 라이브 불필요)

1. **게스트 저장 스킵이 공항 출구에만 적용** — `GameCanvas.jsx` devGuest 전파는 공항
   requestReturn(L2178) 한 곳뿐. 에어허브(L1848)·횡단열차 하차(L2267·L3055)·페리(L2320)는
   저장 실패 시 재시도 토스트만 있고 게스트 스킵이 없어 **게스트 검수로는 대륙 항공·회랑·페리
   이동 불가**. 제품(로그인) 경로는 재시도 UX가 있어 소프트락은 아님. → E7 발주.
2. **게스트 채팅 카피 부정직** — `WorldPage.jsx` L260·L300: 게스트는 영구 '연결 중…' 표기.
   비로그인엔 '로그인하면 이야기할 수 있어요'가 정직. (1줄 카피 — Claude 후속)
3. **미니맵 factor 초과(오너 픽 대기)** — 도쿄 40.96 MiB(계약 대비 170.7%)·코트다쥐르
   27.18 MiB(113.2%). A/B 시각자료 완비(`docs/proposal-minimap-factor-ab.md`) — 픽만 남음.
