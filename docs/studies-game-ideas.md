# 지역학 → 게임 반영 아이디어 보드 (오너 픽 대기)

지역학(스터디즈) 문서를 저작하며 발굴한 게임 반영 후보를 10점 만점으로 평가해 상시 리스트업한다.
점수 = 학습 가치 × 구현 용이성 × 기존 시스템 정합의 종합. **전 항목 owner-gate** — 오너가
픽하면 해당 항목만 착수한다(지역학 1기 "게임 적용 보류" 원칙 유지). 저작 중 발굴분을 계속 추가.

## 9점대 — 기존 인프라 재사용으로 즉시 가능, 효과 큼

### ① NPC·도어 대화에 「왜?」 딥링크 — 9/10
콘비니 NPC의 접객 경어 장면에 "왜 이런 말투를 쓸까?" 버튼 → `/studies/japan/jp-life`의
경어 섹션으로 딥링크. 면세 카운터 → jp-economy 소비세 섹션, 료칸 도어 → jp-culture 온천·료칸 섹션.
- 근거: 학습 순간에 배경 지식을 무는 것이 지역학의 존재 이유 그 자체. 문서에 이미 게임 접점을
  역방향으로 명시해 둠(경어→콘비니 NPC 등) — 순방향 링크만 달면 양방향 완성.
- 구현: 도어/NPC 노드에 `studiesRef: { country, slug, section }` 필드 + 오버레이에 링크 버튼 1개.
  라우트·문서 앵커(sec-N) 기존재. 엔진 무변경.

### ② 지역학 도어 — 박물관 노드의 재발견 — 9/10
"외관만 담아요"로 끝나는 박물관·기념관 POI(뮤셈·우에노 박물관·해양박물관 등)에 3번째 도어
유형(지역학 도어)을 배선 — 열면 관련 지역학 문서로. 전시물 IP 재현 없이 노드가 살아난다.
- 구현: 언어 도어의 track 라우팅 선례 그대로(`studiesRef`), 죽은 링크 방지 계약 테스트 동반.
- 유의: 나라별 문서가 채워진 곳부터(일본 6편 완간 — 일본 도시 우선).

## 8점대 — 소규모 UI 추가로 가능

### ③ 도시 desc 「더 알아보기」 — 8/10
POI desc 박스 말미에 관련 문서 섹션 링크(간다묘진 → 마쓰리 섹션, 동래읍성 → 임진왜란 섹션).
①의 desc 판. 노드당 선택 배선이라 점진 적용 가능.

### ④ 스탬프 지식 카드 — 8/10
스탬프 수집 순간 해당 장소의 지역학 마이크로 팩트 1줄 토스트("에도는 18세기 인구 100만의
세계 최대급 도시였어요"). 수집 보상의 체감 가치 상승.
- 구현: 노드별 factLine 필드(지역학 문서에서 발췌·검증 재사용). 스탬프 우주 확장(오너 보류
  중인 스탬프 확대 결정과 묶어 판단 권장).

### ⑤ 도시 진입 브리핑 카드 — 8/10
나라의 첫 도시 진입 시 1회, 개관 문서 요약 3줄 카드("일본에 오셨어요 — …") + 문서 링크.
여행 서사 강화. 세션당 1회 노출 제어 필요.

## 7점대 — 새 저작·중간 스코프

### ⑥ 지역학 확인 퀴즈 — 7/10
문서 말미 3문항 퀴즈. 기존 refQuiz(문법 퀴즈 빌더) 인프라 재사용 가능성 높음.
- 유의: 문서 스키마에 quiz 필드 추가 — 저작 부담 소폭 증가.

### ⑦ 임진왜란 교차 서사 퀘스트 — 7/10
동래읍성(부산)과 규슈 도자기 문화(후쿠오카권 desc)를 잇는 2도시 방문 퀘스트 — 같은 사건의
양국 흔적을 걸어서 잇는다. 지역학 교차 정합(일본학·한국학 역사편 상호 일치)을 게임 서사로 승격.
- 유의: 퀘스트 시스템 스코프 확인 필요(기존 reviewQuest 구조 재사용 가능성).

### ⑧ 하카타벤 액트 씬(후쿠오카) — 7/10
jp-life 방언 문서에서 도출 — 자갈치 사투리 축([ko/local/gloss]) 문법을 일본 방언에 첫 적용.
후보 장면: 야타이(포장마차) 저녁. 액트 씬 4호 후보로 자갈치·후지 선례 그대로.

## 6점 이하 — 보류 성격(스코프 크거나 효과 불확실)

### ⑨ 계절 연중행사 이벤트 — 6/10
결정적 월드 시계로 하나미·오본·마쓰리 시즌에 도시 프롭·desc 가변. 몰입 효과 크나 엔진 스코프
큼(조수 RFC와 같은 급) — 조수 시스템 오너 결정과 묶어 판단 권장.

### ⑩ 여권(앨범)에 나라 페이지 — 6/10
스탬프 앨범에 나라별 페이지(방문 도시·지역학 링크·국가 개요). 앨범 UI 개편 동반.

### ⑪ 경제 지리 오버레이(관리자 뷰어) — 4/10
뷰어에 산업 클러스터 표시. 학습자 노출 아님 — 우선순위 낮음.

평가 기준 메모: 9점대는 "이번 주말에 켤 수 있고 켜는 즉시 학습 루프가 좋아지는 것",
7점대는 "저작·설계 1라운드 필요", 6점 이하는 "다른 오너 결정과 묶여 있음".
