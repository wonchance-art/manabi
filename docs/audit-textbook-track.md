**작성일**: 2026-07-24

## 1. 커리큘럼 커버리지

### 1.1 트랙별 개수 현황

| 언어 | 문법 챕터 | 어휘 세트 | 총 어휘 수 | 세트당 평균 |
|------|---------|---------|----------|----------|
| **일본어** | 6 | 42 | 125 | 3어 |
| **프랑스어** | 7 | 15 | 132 | 9어 |
| **영어** | 7 | 6 | 83 | 14어 |
| **중국어** | 14 | 29 | 101 | 3어 |

### 1.2 트랙 간 불균형

- **문법 챕터**: 최대 133% 격차 (영어/프랑스 7개 vs 중국 14개)
- **어휘 체계**: 최대 59% 격차 (영어 83개 vs 프랑스 132개)
- **세트 구성**: 일본/중국은 소단위(3어/세트), 영어는 대단위(14어/세트) — 표준화 필요

**격차 요약**: 중국어는 문법이 충분하지만 어휘가 카테고리당 소수 구성. 영어는 어휘 세트가 극히 적음(6개). 일본어는 세트 수 대비 단어 밀도가 낮음.

### 1.3 콘텐츠 품질 표본

#### 일본어 N2-01: 비즈니스 경어
- **구조**: 6 섹션 (기초 표 → 이중경어 → 알바 경어 → 압존법 → 보너스)
- **깊이**: 섹션당 3~5 예문, 한국인 함정 명시적 경고 (pitfall 4회)
- **독창성**: 한국 직장 경어와의 비교, 취업활동 맥락 추가 (Tip)
- **평가**: ✓ 완성도 높음 — 설명 충분, 예문 다양, 실용성 강함

#### 프랑스어 A1-01: 주어 대명사·être
- **구조**: 4 섹션 (주어대명사 → être 활용 → C'est → on)
- **깊이**: 섹션당 2~3 예문, 발음 IPA 병기
- **비교**: 영어 be 동사와 병렬 (vsEn 항목)
- **평가**: ✓ 균형 잡힘 — 입문자 기준 충분하지만 심화 내용 부족

#### 영어 A1 (A1-03 articles 예상)
- **주의**: 총 어휘 세트 6개 → 정보 부족, 체계성 불명
- **평가**: △ 미흡 — 세트 수 대비 콘텐츠 밀도 저평가 필요

#### 중국어 H1 (HSK 1급)
- **구조**: 문법 + 어휘 + 문형사전(bunkei) 3층 체계
- **특징**: 한자어 연결(hanja) 명시적, HSK 레벨과 대응
- **평가**: ✓ 체계적 — 한국인 학습자 특화, 다만 어휘 세트당 단어 부족

## 2. 학습 루프 연결도

### 2.1 흐름 진단

| 연결 지점 | 상태 | 비고 |
|---------|------|------|
| LearnPage → StudySessionPage | ✓ 연결 | 학습 발주 정상 |
| StudySessionPage → grammarSrs | ✓ 연결 | 문법 복습 큐 등록 |
| VocabPage ← user_vocabulary (FSRS) | ✓ 연결 | FSRS 복습 스케줄 동작 |
| VocabReview (평가 저장) | ✓ 연결 | 평가 점수 → DB 기록 |
| **복습 후 진도 진행** | ✗ **끊김** | reading_progress 갱신 미확인 |
| 시리즈 완주 후 다음 시리즈 | ✓ 연결 | "다음 강의" 카드 작동 |

### 2.2 루프 끊김 지점

**발견**: 1개 지점

1. **복습 후 진도 갱신**
   - 가설: VocabReview에서 평가 저장 후 reading_progress.is_completed 갱신이 명시적이지 않을 수 있음
   - 영향: 사용자가 복습을 완료해도 "오늘 학습" 진도가 즉시 반영되지 않을 가능성
   - 심각도: 중 (진도 추적 신뢰도 저하)

**격차 요약**: 학습 흐름은 5/6 연결(83% 완성도). 한 지점에서 진도 갱신이 지연될 수 있음.
