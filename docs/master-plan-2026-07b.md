## 1. 축 A — 월드 × 학습 융합 (이번 기획의 본론)

오너의 원래 선언: "학습 월드에 언어 관련 구축하기 **이전에** 친구들과 모여 이야기 나누며 여행 다닐 수준으로" — 그 전제가 이번에 완성됐다. 이제 "언어 관련 구축"의 차례이고, 이미 가진 부품들이 정확히 맞물린다.

### A-1. NPC 도트 대화 — 배운 일본어를 "쓰는 곳" (킬러 피처 후보)
- 월드의 장소마다 도트 NPC(라멘집 주인·역무원·미코상)를 두고, **포켓몬 대화창(이미 구현됨)으로 일본어 대화**.
- 상호작용은 챕터에서 배운 실전 문장 그대로: 라멘집에서 「替え玉ください」, 이자카야에서 「とりあえず生で」, 신사에서 오미쿠지 뽑기(大吉/凶 리추얼).
- 입력 방식 2단: ① 선택지(배운 표현 3개 중 고르기 — 초보) ② 도트 입력창에 직접 타이핑(정답 판정은 기존 StoryCheck accept 로직 재활용).
- **콘텐츠·판정·UI 전부 기존 자산 재조합** — 신규 인프라 거의 0. 문화 도어 챕터(편의점·이자카야·라멘·신사·전철·온천·신칸센)와 1:1 대응.
- 친구와 함께 서 있으면 NPC 대화가 서로 보임(채팅 로그 공유) → "같이 주문해보기" 자연 발생.

### A-2. 독해 트랙 ↔ 월드 아크 일치 — "읽은 여정을 걷는다"
- 독해 글 4~30 재작성(서사체) 시 각 글의 무대(하네다→시나가와→…)를 **월드 노드와 명시 연결**: 글을 끝내면 해당 월드 장소에 "민준의 발자국" 마커 + 그 장소 NPC가 글 속 대사를 사용.
- 역방향: 월드에서 그 장소에 처음 가면 "이 장소의 이야기 읽기" 도어 → 독해 트랙 해당 화로.
- 스탬프(이미 구현)를 이 연결의 화폐로: 독해 완료 스탬프 + 방문 스탬프가 같은 앨범에.

### A-3. 스탬프 앨범 + 여행 일지 (소형)
- GBC 셸 SELECT 메뉴에 도트 앨범: 모은 스탬프·방문 기록·NPC 대화 달성. 여권에 도장 찍히는 감성.

### A-4. 월드 회화 미션 (친구용, 중형)
- 주간 미션: "친구와 신사에서 만나 おみくじ 뽑고 결과를 채팅으로 말하기" — 채팅 로그에서 일본어 사용 감지(가벼운 판정) 시 보상 스탬프.

**추천**: A-1 파일럿(라멘집·신사 2곳) → A-2는 독해 글 재작성과 동시 진행 → A-3 → A-4.

## 2. 축 B — JA 콘텐츠 심화 (광맥 소진 + 어휘 보강)

### B-1. 어휘 총량 보강 (진단 근거: JA 981 = 전 언어 최하, 문형 936과 기형적 불균형)
- FR(FLELex)·ZH(HSK)에서 검증된 "오픈소스 골격+생성" 방법론을 JA에 이식: JLPT 공개 어휘 리스트 골격 + 급별 생성 웨이브. 목표 1차 +800(N5·N4 두텁게 — 생초보 정책과 일관).
- 기존 재미 테마(속어·문화)는 유지, 이번엔 **기본 어휘의 두께**.

### B-2. 문화 도어 신작 (광맥 3·4·7 — 시의성 순)
1. **면세 쇼핑 도어(최우선)** — 2026-11-01 리펀드 개편 선제 대응(정부 공식 근거 확보됨). 시기 놓치면 가치 급락.
2. 벚꽃 유닛(3~4월 방일 피크 대비) — 桜前線·場所取り·花より団子.
3. 식당 은어 실전(つゆだく·あがり·むらさき) — 어휘 반영분을 챕터 도어로 승격.
4. 가챠·J-호러(8番出口)·치이카와 — 젊은 층 훅.

### B-3. 한자음 스킬트리 (광맥 7 톱2 — 구조 신설)
- "받침 ㄱ→く/き" 같은 규칙 1개가 어휘 수십 개를 해금하는 한국어 화자 전용 치트키. 챕터 1~2개 + 어휘 테마 연동. KCI 논문 근거 확보됨.

### B-4. 독해 트랙 완주 + 차기 트랙
- 글 4~30 서사체(하나씩 승인 → 검증 후 3~5편 배치 전환 제안). 완주 후 차기 트랙 "오사카/교토"(간사이 문화 도어들과 연결).
