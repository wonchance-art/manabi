# RFC — 여행책 지구제 (Guidebook Districts)

- 상태: 설계 확정 제안(오너 발제 2026-07-23 반영). 구현 발주: D1~D4(#150).
- 발제 요지: 도시맵을 여행책의 미니멀 도시맵처럼 — 핵심 지리 골격 + 이동성 우선 도로,
  동네(지구) 단위 단계 완성. 완성 구역만 이동 개방, 미완성 구역 이동 제한.

## 1. 기존 26도시 맵 처리 판정: **폐기하지 않는다 — 게이팅 자산으로 전환**

| 옵션 | 비용 | 리스크 | 판정 |
|---|---|---|---|
| 전면 재생성(간소 맵 새로 그림) | 수집·본생성·verifier 26도시 재주행(수 주) | 결정성·해시 게이트 자산 소실, 지리 정확도 후퇴 | ✗ |
| **기존 geo + 지구 게이팅 레이어** | 엔진 1~2주(세션 병렬) + 지구 마스크 저작 | 낮음(fail-closed) | **✓ 채택** |

근거: 오너 제안의 실체 4요소(핵심 지리 골격·이동성 도로·단계 완성·미완 제한)는 전부
**geo 위 레이어**로 달성된다. 현행 geo가 이미 "큰 지리 골격"(강·해안·간선·POI 실좌표)을
보유 — 여행책화의 원료다. v3 mainRoute(포장·이정표·발견)가 첫 레이어였고, 지구제는 그
일반화다. 부팅 lazy-load(P1)·미니맵 factor와도 시너지(개방 지구만 고밀도).

## 2. 데이터 계약 (도시 파일 저작 핀 — fail-closed)

```js
districts: {
  version: 'district-v1',
  open: [
    { id: 'presquile', label: '프레스킬', tiles: { rects: [[x0,y0,x1,y1], …] } },
  ],
  locked: {
    style: 'guidebook',
    line: '이 동네는 아직 준비 중이에요 — 다음 여행에서 만나요.',
  },
}
```

- `districts` 미정의 도시 = 전면 개방(렌더·이동 현행 불변). 정의 시에만 게이팅.
- `open[].tiles`는 축정렬 rect 합집합(결정적·저작 쉬움). 폴리곤은 v2로 유보.
- 개방 판정: 타일이 어느 open rect에도 없으면 잠김. TRANSIT 도착·스폰·returnNode가
  잠김 타일이면 **계약 위반으로 resolve 단계 throw**(정합 게이트).

## 3. 동작 계약

- **이동 제한(soft wall)**: 잠김 타일로의 보행 스텝을 개방 경계에서 정지시키고 GBC 말풍선
  1회(4.2s, `locked.line`). 도로·지형은 보이므로 "지도에 있지만 아직 안 가 본 동네" 문법.
- **여행책 렌더(guidebook tone)**: 잠김 지구는 지형·수면·간선도로·랜드마크 실루엣만 남긴
  간소 렌더(채도 낮춘 종이 톤, 건물 디테일·프롭·NPC·발견 스팟 생략). 성능 절약 동반.
- 미니맵·여행 수첩: 개방 지구 채색, 잠김 지구는 윤곽선+빗금. 도시 상세에 '개방 n/m 동네'.
- 발견 이벤트·도어·NPC 배선은 개방 지구에만 존재해야 한다(정합 게이트에 포함).

## 4. 적용 순서 (콘텐츠 공백 실측 기반 — audit-static-sweep §B)

1. **리옹(파일럿)**: mainRoute 회랑(프레스킬~파르디외 띠) = open, 나머지 잠금.
   이미 포장·발견 8·카드가 그 회랑에 있어 "완성 동네" 즉시 성립.
2. **보르도·스트라스부르(빈 도시)**: 역~중심 코어 지구만 open — 빈 벌판을 잠그고
   T3 후보 경로(docs/proposal-mainroute-bdx-sxb.md) 위에 콘텐츠를 지구 단위로 채움.
3. 희박 대형맵(서울·코트다쥐르·부산·레만): 코어+명소 지구 open, 외곽 잠금.
4. 신규 도시: 수집·본생성·verifier는 전역 유지(계약 불변), 배선·개방은 코어 지구부터.

## 5. 분업·발주

- **D1(Codex-1)**: 엔진 — districts resolve·soft wall·guidebook 베이크·정합 게이트·테스트.
- **D2(Claude)**: 지구 마스크·잠금 카피 저작(리옹→보르도→스트라스부르), 지구 단위 콘텐츠 계획.
- **D3(Codex-3)**: 미니맵·여행 수첩 지구 표기(개방률).
- **D4(Codex-4)**: 잠김 지구 청크·베이크 절약 실측(도쿄 factor 트랙과 파일 경계 분리).

## 6. 오너 확인 포인트 (진행 중 병행 — 블로킹 아님)

1. guidebook 톤 시안 1장(D1이 리옹 잠금 지구 스크린샷 첨부) — 톤 확정.
2. 잠금 카피 기본안: "이 동네는 아직 준비 중이에요 — 다음 여행에서 만나요." 수정 여부.
3. 적용 순서 §4 승인(이의 없으면 그대로).
