# 오버월드 지역 스트리밍 런타임

## 범위

| 지역 | 씬 ID | 크기 | 횡단철도 게이트 |
|---|---|---:|---|
| 아시아·태평양 | `overworld:asia-pacific` | 2631×2669 | 블라디보스토크 `(1576, 442)` |
| 유럽·지중해·중동 | `overworld:emea` | 964×1137 | 모스크바 `(768, 253)` |

- 두 지역 모두 `releaseEligible=false`다.
- 체크인된 playability preview `.owc`를 256×256 저장 청크 단위로 읽는다.
- 화면에는 카메라 주변 32×32 렌더 페이지만 유지하며 기존 오버월드 LRU 상한을 적용한다.
- 표면·충돌·view-only 판정은 같은 청크에서 읽는다. 바다·충돌·view-only 타일은 이동할 수 없다.

## 저장과 복귀

1. 지역 안 보행 좌표는 기존 `world_positions(scene, x, y)`에 저장한다.
2. 재접속 시 등록된 씬과 지역 범위 안 좌표만 복원한다.
3. 저장 타일이 실제로 보행 불가하면 해당 지역의 횡단철도 게이트로 폴백한다.
4. 지역에서 횡단철도로 이동할 때 플랫폼을 먼저 저장하고, 저장 성공 뒤 씬을 전환한다.
5. 횡단철도에서 지역으로 이동할 때 지역 게이트를 먼저 저장하고, 저장 성공 뒤 씬을 전환한다.

## 현재 제한

- 지역 콘텐츠는 아직 없다. KR/JP 외 지역은 지형·교통만 제공한다.
- 강·철도는 카메라가 걸친 256타일 저장 청크의 JSON만 불러와 화면 범위 선분만 그린다.
- 1타일 halo의 중복 선분은 전역 segment ID로 제거하며, 레이어별 최근 12개 문서만 메모리에 둔다.
- 국경·지명 표시는 민감지역 정책과 언어 앵커 검수 뒤 별도 통합한다.
- 장거리 AOI 계약 전까지 다른 이용자는 지역 씬에서 표시하지 않는다.
- 유럽 지역의 최종 투영·민감 경계 판정 전에는 출시 대상으로 승격하지 않는다.
