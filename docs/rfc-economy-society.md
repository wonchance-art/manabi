# [DESIGN RFC] 경제·사회 시스템 — 화폐/부동산/자치/직업 (2026-07-15, 오너 기획 세션 수렴)

Codex 핑퐁용. 구현 지시 아님 — 경제 시뮬·원장 설계·어뷰징 방어 관점의 반론/대안/작업량 요청.
관통 원칙: **모든 화살표가 학습으로 수렴** + **현실 정보 융합** + **폐쇄 경제**(실화폐 금지).

## 기둥 ① 경제 (화폐·수입)
- 단일 게임 화폐, 서버 원장(지갑·거래 전부 서버 권위 — 스탬프와 정반대, A-4 원칙).
- 수입: 학습(챕터·SRS·독해·문화도어) 기본 수입 + 직업(기둥④) + 티칭(첨삭·답변, 피어평가 게이트).
- 지출(sink): 부동산·건축·장식·교통(?)·동네 금고. 발행/소각 시뮬 필수(Codex).
- **현실 물가/급여 반영(오너)**: 일본 후생노동성 현별 최저임금·국토교통성 지가공시, 한국 최저임금위·
  공시지가 — 버전 고정 데이터 파일+출처 manifest(연 갱신 = 현실 연동 라이브 이벤트).
  **순위·체감 현실 유지, 스프레드 로그 압축(예 30~50:1 캡)** — 밸런스 vs 현실성의 유일 긴장.
  통화 표기: 내부 단일 + 지역 표기 ¥/₩(고정 환산). 실환전 v2 검토.
- 어뷰징: 파밍 봇·자문자답 티칭 → 활동 가중·이상치 탐지·일일 상한.

## 기둥 ② 부동산 (구획·하우징)
- **하이브리드**: 지도 비기능 블록 일부를 분양 구획(plot)으로 지정(유한·동네별 실지가 기반 가격),
  내부는 인스턴스 씬(CityScene 파라미터화 재사용). 전 타일 자유 구매 금지.
- 보유 상한 + 보유세(sink). 초대(멀티 씬별 분리 재사용).
- **지도 안정화 게이트**: 좌표 재생성이 끝난 맵부터 분양(도시맵 먼저, 오버월드는 ① 재투영 후).
  소유권 레코드는 tile 이 아니라 **plot id**(재생성 시 재매핑 가능하게).

## 기둥 ③ 자치 (동네 거버넌스)
- ZONES = 선거구(도쿄 9·후쿠오카 7 기존). 주민 = 구획 소유+활동 학습자.
- v1 이장 자원제(활동 요건+운영 승인) → 인구 임계 후 선거(시즌 4~6주·소환·활동 미달 자동 공석).
- 권한 티어: 큐레이션·행사(게시판·이벤트 소집·장식 프리셋·추천 핀) = v1 /
  경제(동네 금고 운용) = v2 / **모더레이션·콘텐츠 사실 = 위임 금지(플랫폼 소유)**.
- 감사 로그 + 플랫폼 오버라이드 상시. 투표 서버 검증·활동 가중(부계정 방어).

## 기둥 ④ 직업 (플레이어 노동)
- **NPC 폴백 + 플레이어 승격**: 기본은 NPC 스크립트/보이스 운행(현 NPC 자산 = 직무 대본 원형),
  플레이어가 그 역을 실연으로 승격. 월드는 절대 비지 않음.
- 자격 = 챕터 수료(ot-07 수료 → 편의점 알바). 배우기→자격→일하기→벌기→집 루프.
- 무대: 편의점·이자카야·라멘집·역 안내·열차/비행 승무(직접 하차 교통 = 시프트)·동네 가이드(자치 연계).
- 수행: v1 텍스트(대본+선택지+제한 자유입력), 음성은 고신뢰 후. 산출(production) 연습의 게임화 —
  敬語·서비스 표현. 고객측은 듣기 연습(양방 학습).
- 급여 = 기본급(시프트 완수, 지역 최저임금 기반) + 평가 보너스 + 팁(유저 간 이전·인플레 중립).
- 평가: 별점+정형 태그. 가중·상호보복 탐지·저표본 보호. 자유발화 모더레이션은 플랫폼(필터+신고).

## 단계 (확장 아크와 인터리브 — 확장 중단 없음)
P0 본 RFC 핑퐁 → P1 지갑+학습 수입(DB 오너 승인) → P2 부동산 v1(안정화 도시맵 구획) →
P3 하우징 인스턴스+초대 → P4 직업 v1(편의점·열차 승무 텍스트) → P5 자치 v1(이장) → 이후 선거·금고·티칭 마켓.

## Codex 에게 묻는 것
A. 발행/소각 시뮬 초안 — 학습 수입률·직업 기본급·부동산 가격대·보유세로 인플레 중립이 되는 파라미터 골격.
B. 원장 스키마 개요(지갑·거래·소유권·plot·시프트·평가) — 하드리밋 승인용 초안. plot id 재매핑 전략.
C. 평가/티칭 어뷰징 방어 설계(가중·탐지·상한)의 실무 구조.
D. 실물가 데이터 계약 — 최저임금·지가공시 인제스트 파이프라인(버전·출처 manifest·연 갱신), 로그 압축 함수 제안.
E. 각 단계 작업량 추정 + 확장 아크와의 리소스 간섭 평가.
