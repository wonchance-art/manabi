# 제품 정의 (F0) — 오너 승인 대기 초안

- 상태: **초안 — 오너 승인 항목**(승인 시 이 줄을 확정 표기로 갱신).
- 관계: docs/plan-foundation.md의 F0. 모든 코스·레슨 설계(F1~)의 뿌리.

## 제품 한 줄

한국어 사용자 성인 초학습자가 **하루 15~20분, 4주**에 한 언어의 "여행 생존 회화 + 기초 문법"을
완주하는 학습 웹앱. (4트랙 균등: 일본어·프랑스어·영어·중국어 — 주력 없음, 오너 결정 2026-07-24)

## 레벨 1 완주 정의 (측정 가능)

완주자는 다음 **4장면 × 약 40문형**을 보고 이해하고, 빈칸·선택 연습에서 80% 이상 정답:

| 장면 | 대표 과업 | 문형 수(안) |
|---|---|---|
| 공항·이동 | 입국 절차 응답, 표 구입, 환승 질문 | ~10 |
| 숙소 | 체크인/아웃, 요청(수건·와이파이), 문제 신고 | ~10 |
| 식당·상점 | 주문, 계산, 추천 요청, 알레르기/제외 요청 | ~10 |
| 길·긴급 | 길 묻기/알아듣기, 교통수단, 아픔·분실 표현 | ~10 |

+ 트랙별 문자·발음 기초(일본어 가나·한자 소량 / 중국어 성조·병음 / 불어 발음 규칙 / 영어 강세).

## 코스 형태

- Course(트랙×레벨) → Unit(주차 테마 = 위 4장면) → Lesson(15~20분: 문법 개념 + 어휘 세트 + 연습)
  → Review(FSRS 큐). 레벨 1 = 4 Unit × 주당 5 Lesson 내외(트랙 콘텐츠 실측에 맞춰 조정).
- 완주 산출: 코스 지도에서 레벨 1 전 레슨 완료 + 복습 큐 소화율 표시(F5).

## 대상·전제

- 대상: 한국어 UI, 성인 초학습자(해당 언어 학습 경험 무전제). 게스트로 전 기능 체험 가능(로그인 시 동기화).
- 비목표(레벨 1 범위 밖): 발화 평가·청해 오디오(owner-gate 후보), 비즈니스 회화, 시험 대비 전문 과정.

## 측정

- 완주율(레벨 1), 주간 복습 소화율, 레슨당 평균 소요 시간(15~20분 검증), 연습 정답률(80% 기준).
