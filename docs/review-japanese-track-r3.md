## 1. 결론과 재등급

### 종합 등급: C → C

R2 뒤 [PR #619](https://github.com/wonchance-art/manabi/pull/619)이 두 신규
Important를 직접 수리했다. 한자 레벨 검사는 실제로 오류 7→0, 요미가나 실패
0으로 복구됐다. 알바 경어도 `〜になります`와
`よろしかったでしょうか`의 형식 자체를 X로 두던 표제·본문을 장면과 기능의
부적합으로 범위화하고, 실제 변화와 과거 재확인은 정상이라는 반례를 넣었다.

그러나 JA-R2-I01은 완전히 닫히지 않았다. 같은 섹션 마지막 tip은 취업 면접과
전화 응대에서 두 형식이 “입에서 나오는 순간” 알바 경어 인상을 준다고 다시
형식 단위로 단정한다. 앞의 범위화와 충돌하므로 Important 잔여로 본다.

그 밖의 R2 자연성 표본, N4·N3·N2 문형 목표 미달, 비공식 JLPT 목록·생성
예문의 출처 표지, 상호·작품·캐릭터 무재현 규약 충돌은 #616~637에서
수리되지 않았다. 따라서 구조·검증은 C→B로 회복했지만 종합 B 합격선은 아직
충족하지 못해 C를 유지한다.

6. 콘텐츠 lint·한자/요미가나 검사·일본어 targeted Vitest·전체 Vitest와

확정 근거로 등급에 새로 올릴 R3 문제는 없다.

`n2.js:73`의 tip은 #619 변경부 밖에 남은 JA-R2-I01의 동일 범위 문제이므로
신규 ID를 만들지 않고 이월 항목의 부분 수리로 처리했다. 변경 4파일과 R2
고위험 축에서 그 밖의 후보도 검토했지만, 새 Critical 또는 Important로
확정할 근거는 찾지 못했다.

## 6. 평가축별 재판정

### 언어 정확성 — C 유지

#619의 pattern/body는 형식 자체와 부적절한 장면을 구분한다. 이 변화로
JA-R2-I01의 중심 오개념은 상당 부분 줄었다. 그러나 같은 섹션의 마지막 출력
tip이 두 형식을 문맥 없이 다시 낙인찍어 학습 규칙이 내부 충돌한다. 현재 최고
심각도는 Important이고 제한적 교정이 필요하므로 C를 유지한다.

### 발음·읽기 — B 유지

R2에서 수리된 모라 정의와 대표 음성 표기는 회귀하지 않았다. #619의 가나화와
예외 등록 뒤 `scripts/check-content.mjs`의 요미가나 검사는 53파일 중 실패
0이다. 새 감점이나 상향 근거가 없어 B를 유지한다.

### 커리큘럼 정합 — C 유지

문형 수는 N5 125, N4 159, N3 195, N2 209, N1 248로 같다. N4·N3·N2의
목표 미달과 R2의 레벨 간 중복 백로그, 비공식 JLPT 목록·생성 예문의 출처
정책은 #619 범위가 아니다. 중요 구조 백로그가 남아 C다.

### 한국어 설명 — B 유지

R2와 같은 정규식으로 번역 `ko`를 제외하면 비번역 설명 영역 격식체는
`grammar/n3.js`의 `-습니다` 1건이다. 해당 문구와 다른 설명은 #619에서
바뀌지 않아 B를 유지한다.

### 예문 자연성 — B- 유지

### C → C의 의미

  부분 상태다.

못한 C다.
