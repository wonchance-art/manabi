## 2. 이행 방식

1. 스크립트: 표제 zh 글자 수와 병음 음절 수 대조 → 2~3자 단일어 후보 자동 연서,
   4자는 2+2, **성어·이합사·고유명사 후보는 보류 목록**으로 분리.
2. 보류 목록(예상 수백 건)은 표본이 아니라 **전수 수동 판정**(사전 대조).
3. 배치 PR: 팩 단위(h1~h6), 회당 자동분+판정분 묶음. 각 배치에 무작위 50건
   사전 대조 표본 검수 기록.
4. 게이트: 병음 문자 집합 불변(성조 부호 손실 0)·항목 수 불변·전체 vitest.
